# Landing Page for Daily Activities

## Features
- Currency exhange rate comparison
- ToDo List
- News RSS Aggregator
- Encrypted Notes
- Calendar

#### Current Dev-state ~ very alpha

## Requirements
- Docker
- Web Browser

## Instructions

### 1. Clone This Repo
```bash
git clone https://github.com/entropicvail/vite-daily-landing.git
```

### 2. Spin-up App in Docker
```bash
cd vite-daily-landing/vite-todo-list
```
```bash
docker compose up
```

### 3. Access Services

**Services Available**

*Front-end (website)*
- https://ui.localhost
- http://localhost:5173

*Back-end (express api)*
- https://api.localhost
- http://localhost:3001

### 4. Close Program
```bash
docker compose down
```
